# Walmart Sales Data Cleaning Project

This project focuses on cleaning and preparing a Walmart sales dataset using Python and Pandas.

## Objectives
- Handle missing values (drop / fill)
- Fix incorrect data types
- Remove duplicate records
- Standardize column names
- Export a cleaned dataset

## Files in This Repository
- `project.ipynb` – Jupyter Notebook with all cleaning steps  
- `Walmart.csv` – Original raw dataset  
- `walmart_clean_data.csv` – Cleaned dataset  

## Tools Used
- Python  
- Pandas  
- Jupyter Notebook  
- VS Code  

## Cleaning Steps Performed
1. Checked for missing values  
2. Converted data types (dates, numbers)  
3. Removed duplicates  
4. Renamed columns to lowercase  
5. Saved the cleaned dataset  

## Output
A cleaned and structured dataset ready for analysis.
